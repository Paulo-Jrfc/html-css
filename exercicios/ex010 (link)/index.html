Usando link externos

Você pode acessar o meu reporitório publico no GitHub

Você também pode acessar o meu canal gratuito no YoutTube

E assim ficou o código:

        
            <p>Você pode acessar o meu <a href="https://gustavoguanabara.github.io" target="_blank" rel="external">reporitório publico no GitHub</a></p>
            <p>Você também pode acessar o meu <a href="https://youtube.com/cursoemvideo/">canal gratuito no YoutTube</a></p>

As tags( taget="_blank" e a tag rel="external" ) dentro da tag <a href="">servem para redirecionar um link externo para uma nova guia, permitindo que o seu site ainda fique aberto para que o cliente consiga voltar ao seu site.
Podemos ver a diferença abrindo os sites, o site do GiTHub tem essa função ativada, já o site do Youtube não tem essa função ativada.

Usando link internos

Esta é a primeira página do site. Se você quiser pode também acessar a segunda página do meu site

O código para links internos fica assim:

    
        <a href="pag002.html" rel="next">segunda página</a> do meu site</p>

Ao usarmos a tag (rel="next") dentro da tag <a href=""> estamos informando ao google qual o sistema de navegação que estamos criando no site. (next=posterior ou próxima página)

Visualmente isto não interfere em nada.
