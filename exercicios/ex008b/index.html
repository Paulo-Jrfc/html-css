O comando document.gotElementById('teste') é escreito em linguagem JavaScript (Fonte mono espaçada)

num = int(input('digite um número')) if num% 2 == 0 print(f'O número {num} é PAR) else: print(f'O número {num} é ÍMPAR) print('Fim do programa')

O código acima não tem indentaçao porque foi usado a tag <code>

        
            num = int(input('digite um número'))
            if num% 2 == 0
                print(f'O número {num} é PAR)
            else:
                print(f'O número {num} é ÍMPAR)
            print('Fim do programa')
O código acima tem indentação porque foi usado a tag <pre> antes da tag <code>

Este é o código completo:

        
            <pre>
                <code>
                    num = int(input('digite um número'))
                    if num% 2 == 0
                        print(f'O número {num} é PAR)
                    else:
                        print(f'O número {num} é ÍMPAR)
                    print('Fim do programa')
                </code>
            </pre>

O código <code>; deixa na fone 'mono espaçada', que é uma ótima fonte para se utilizar quando se quer mostrar um código.

O código <pre> deixa com indentação na visão do cliente.
