Listas ordenadas

  8. Voltar para casa
  9. Jantar

Podemos usar a tag <li>, para uma melhor organização da nossa lista veja o código a seguir:

 
        
                <ol type="1" start="1">
                    <li>acordar</li>
                    <li>Ligar para o João
                    <li>Tomar Café
                    <li>Escovar os dentes
                    <li>Ir para a faculdade
                    <li>Almoçar
                    <li>Ir para o trabalho
                    <li>Voltar para casa
                    <li>Jantar
                    <li>Dormir
                </ol>

Temos algumas opçoes para organização do código, atravez do (type) que esta dentro da tag <ol> podemos escolher as seguintes opções:.

A=Alfabeto em letras maiusculas

a=Alfabeto em letras minusculas

1=sequencia númerica

I=Números romanos maisculos

i=Números romanos minusculos

Com a tag <start> dentro da tag <ol> definimos a partir de qual letra / número deve começar.

Lista não ordenadas

  • Pão
  • Leite
  • Tomate
  • Alface
  • manteiga
  • Arroz
  • Feijão

Para as listas não ordenadas, o código fica assim:

            
                <code>
                    <ul type="">
                        <li>Pão</li>
                        <li>Leite</li>
                        <li>Tomate
                        <li>Alface
                        <li>manteiga
                        <li>Arroz
                        <li>Feijão
                    </ul>
                </code>

Vale lembrar que a tag <li> não precisa ser fechada com a tag </li>

Temos algumas opçoes de layout para o ícone que vem antes do item da lista, veja a segui:

  • disc
  • circle
  • square

Mudamos esse layout atravez do type de dentro da tag ul <ul type="(opção escolhida)">
