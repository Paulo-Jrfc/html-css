Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (a tag <b> é uma tag não semantica)

Nesta frase tenho um termo em destaque usando a tag <strong> (a tag <strong> é uma tag com semantica)

Itálico / Ênfase

Nesta frase temos um termo em itálico, com a tag <i> (a tag <i> é uma tag sem semantica)

Nesta frase temos um termo em ênfase usando a tg <em>(a tag <em> é uma tag com semantica)

Texto marcado

Podemos criar tambem um texto marcado usando a tag & <mark> ;

Podemos criar tambem um texto marcado usando outra cor, com a tag <mark style background-color> ;

Caso queira que todos os <mark> fiquem da mesma cor, mudamos a cor no cabeçalho da seguinte forma:

<style>

mark { background-color: lime }

</style>

Texto grande e pequeno

Texto deletado

Podemos marcar um tetxo como excluido para indicar que ele deve ser lido mas não considerado.

texto inserido
